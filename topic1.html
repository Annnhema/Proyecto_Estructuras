Arrays Data Structures

Información sobre Arrays

Definición:
Un array (arreglo) es una estructura de datos que almacena una colección de elementos del mismo tipo en una secuencia contigua de memoria. Los elementos en un array se pueden acceder mediante un índice o subíndice, que generalmente comienza en 0.

Características:
- Tipo homogéneo: Todos los elementos de un array son del mismo tipo de datos (ej. enteros, cadenas, objetos).
- Índices: Los elementos de un array se acceden mediante índices. En la mayoría de los lenguajes de programación, el índice inicial es 0.
- Tamaño fijo: El tamaño de un array es fijo en lenguajes como C o Java. Sin embargo, algunos lenguajes (como Python o JavaScript) permiten arrays dinámicos.
- Acceso aleatorio: Los elementos de un array se pueden acceder directamente usando su índice, lo que permite un acceso eficiente a los datos.
- Almacenamiento contiguo: Los elementos de un array se almacenan de forma continua en memoria.

Operaciones comunes:
1. Acceso: Obtener el valor de un elemento en el índice i.
array[i]

2. Modificación: Cambiar el valor de un elemento en el índice i.
array[i] = nuevo_valor

3. Longitud: Obtener la cantidad de elementos en un array (en Python o JavaScript).
len(array)

4. Recorrido: Iterar sobre todos los elementos.
for elem in array:
     print(elem)

5. Inserción: Insertar un elemento en una posición específica (algunos lenguajes permiten hacerlo, pero la mayoría de los arrays tienen tamaño fijo).
array.insert(i, valor)

6. Eliminación: Eliminar un elemento en una posición específica.
del array[i]

7. Búsqueda: Buscar un elemento en el array.
if valor in array:
     print("Elemento encontrado")

8. Ordenamiento: Ordenar los elementos en el array.
array.sort()

Ejemplos:
Ejemplo en Python:

# Definición de un array
              arr = [1, 2, 3, 4, 5]
              
              # Acceder a un elemento
              print(arr[2])  # Salida: 3
              
              # Modificar un elemento
              arr[3] = 10
              print(arr)  # Salida: [1, 2, 3, 10, 5]
              
              # Recorrer el array
              for num in arr:
                  print(num)
              
              # Longitud del array
              print(len(arr))  # Salida: 5

Ejemplo en C:
#include <stdio.h>

              int main() {
                  // Definir un array
                  int arr[5] = {1, 2, 3, 4, 5};

                  // Acceder a un elemento
                  printf("%d\n", arr[2]);  // Salida: 3

                  // Modificar un elemento
                  arr[3] = 10;
                  printf("%d\n", arr[3]);  // Salida: 10

                  // Longitud del array (en C, se calcula de forma manual)
                  int length = sizeof(arr) / sizeof(arr[0]);
                  printf("%d\n", length);  // Salida: 5

                  return 0;
              }

Ejemplo en JavaScript:
// Definir un array
              let arr = [1, 2, 3, 4, 5];

              // Acceder a un elemento
              console.log(arr[2]);  // Salida: 3

              // Modificar un elemento
              arr[3] = 10;
              console.log(arr);  // Salida: [1, 2, 3, 10, 5]

              // Recorrer el array
              arr.forEach(num => console.log(num));

              // Longitud del array
              console.log(arr.length);  // Salida: 5

Tipos de Arrays:
1. Arrays unidimensionales: Son los más comunes. Se representan como una lista de elementos.
2. Arrays multidimensionales: Arrays de arrays. Se usan para representar matrices o tablas.
# Array 2D en Python (matriz 3x3)
matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
3. Arrays dinámicos: Algunos lenguajes como Python y JavaScript permiten arrays cuya longitud puede cambiar durante la ejecución del programa.
# Array dinámico en Python
arr = [1, 2, 3]
arr.append(4) # Agregar elemento

Desventajas de los Arrays:
- Tamaño fijo: En lenguajes como C o Java, el tamaño debe ser especificado al momento de la creación.
- Desperdicio de memoria: Si no se utilizan todos los elementos, puede haber desperdicio de memoria.

Comparación con otras estructuras de datos:
- Los arrays tienen un acceso más rápido debido al acceso aleatorio mediante índices.
- Son menos flexibles en comparación con las listas enlazadas o vectores dinámicos, que pueden cambiar de tamaño fácilmente.
