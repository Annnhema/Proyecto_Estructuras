Circular Linked List

Información sobre Listas Enlazadas Circulares

Definición:
Una lista enlazada circular es una variante de la lista enlazada donde el último nodo apunta de nuevo al primer nodo, formando un ciclo. A diferencia de una lista enlazada simple, que tiene un puntero nulo al final, en una lista circular el último nodo mantiene una referencia al primero.

Características:
- El último nodo de la lista está conectado al primer nodo, creando un ciclo.
- No hay un nodo "final" con puntero nulo. La lista puede ser recorrida infinitamente sin llegar a un valor nulo.
- Es útil en situaciones donde se necesita recorrer los elementos repetidamente, como en la implementación de estructuras de datos cíclicas (ej. juegos, aplicaciones de reloj, etc.).

Operaciones comunes:
1. Insertar al principio: Insertar un nuevo nodo al principio de la lista circular.
list.insertFirst(elemento)

2. Insertar al final: Insertar un nuevo nodo al final de la lista circular.
list.insertLast(elemento)

3. Eliminar el primer nodo: Eliminar el primer nodo de la lista circular.
list.removeFirst()

4. Eliminar el último nodo: Eliminar el último nodo de la lista circular.
list.removeLast()

5. Recorrido: Recorrer todos los nodos en la lista circular.
list.print()

Ejemplo de implementación en pseudocódigo:


                class Node {
                    data
                    next
                }

                class CircularLinkedList {
                    head

                    insertFirst(elemento) {
                        let newNode = new Node(elemento)
                        if (this.head == null) {
                            this.head = newNode
                            newNode.next = this.head
                        } else {
                            let current = this.head
                            while (current.next != this.head) {
                                current = current.next
                            }
                            current.next = newNode
                            newNode.next = this.head
                            this.head = newNode
                        }
                    }

                    insertLast(elemento) {
                        let newNode = new Node(elemento)
                        if (this.head == null) {
                            this.head = newNode
                            newNode.next = this.head
                        } else {
                            let current = this.head
                            while (current.next != this.head) {
                                current = current.next
                            }
                            current.next = newNode
                            newNode.next = this.head
                        }
                    }

                    print() {
                        let current = this.head
                        do {
                            console.log(current.data)
                            current = current.next
                        } while (current != this.head)
                    }
                }

                };

                struct CircularLinkedList {
                    struct Node* head;

                void insertFirst(struct CircularLinkedList* list, int elemento) {
                    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
                    newNode->data = elemento;
                    if (list->head == NULL) {
                        list->head = newNode;
                        newNode->next = list->head;
                    } else {
                        struct Node* current = list->head;
                        while (current->next != list->head) {
                            current = current->next;
                        }
                        current->next = newNode;
                        newNode->next = list->head;
                        list->head = newNode;
                    }
                }

                void print(struct CircularLinkedList* list) {
                    struct Node* current = list->head;
                    if (current == NULL) {
                        printf("Lista vacía\n");
                        return;
                    }
                    do {
                        printf("%d -> ", current->data);
                        current = current->next;
                    } while (current != list->head);
                    printf("...\n");
                }

                    }

                        let newNode = new Node(elemento)
                        if (this.head == null) {

Ventajas de las listas enlazadas circulares:
- Permiten recorrer la lista de manera continua, lo cual es útil en ciertas aplicaciones, como en sistemas de "round-robin" (planificación de procesos) y en juegos donde los jugadores deben ser recorridos en ciclo.
- El uso de listas circulares puede hacer que las operaciones de inserción y eliminación sean más rápidas y fáciles de manejar cuando se utiliza el último nodo.

Desventajas:
- El recorrido de la lista puede ser más complicado si no se tiene cuidado con el ciclo.
- Si se olvida actualizar correctamente los enlaces, se puede terminar con un ciclo infinito.

Comparación con listas enlazadas simples:
- La principal diferencia es que en una lista circular, el último nodo apunta de nuevo al primer nodo, mientras que en una lista enlazada simple el último nodo apunta a null.
- Las listas circulares son ideales para casos donde se necesita recorrer los elementos en forma continua sin preocuparse por el final de la lista.
