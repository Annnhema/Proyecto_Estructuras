Two Dimensional Arrays

Información sobre Arrays Bidimensionales

Definición:
Un array bidimensional es una estructura de datos que se puede representar como una tabla o matriz, donde cada elemento es accesible mediante dos índices. Es una colección de arrays unidimensionales, por lo que puedes pensar en él como una "matriz" o "tabla" de filas y columnas.

Características:
- Acceso por dos índices: Los elementos se acceden usando dos índices, uno para la fila y otro para la columna.
- Dimensionalidad: Un array bidimensional tiene una dimensión de 2, es decir, filas y columnas.
- Almacenamiento contiguo: Al igual que los arrays unidimensionales, los arrays bidimensionales suelen ser almacenados de forma contigua en memoria.
- Tamaño fijo: Los tamaños de las filas y las columnas se deben definir al momento de la creación en algunos lenguajes como C o Java, mientras que en lenguajes como Python o JavaScript, pueden ser dinámicos.

Operaciones comunes:
1. Acceso: Obtener el valor de un elemento en una posición [i][j].
array[i][j]

2. Modificación: Cambiar el valor de un elemento en la posición [i][j].
array[i][j] = nuevo_valor

3. Recorrido: Iterar sobre todas las filas y columnas del array.
for i in range(filas):
     for j in range(columnas):
         print(array[i][j])

4. Inserción: Insertar un nuevo array (fila) o un elemento dentro de una fila.
array[i].insert(j, valor)

5. Eliminación: Eliminar un elemento o una fila.
del array[i][j]

Ejemplos:
Ejemplo en Python:

# Definición de un array bidimensional (matriz 3x3)
              matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
              
              # Acceder a un elemento
              print(matriz[1][2])  # Salida: 6
              
              # Modificar un elemento
              matriz[0][1] = 10
              print(matriz)  # Salida: [[1, 10, 3], [4, 5, 6], [7, 8, 9]]
              
              # Recorrer el array bidimensional
              for i in range(len(matriz)):
                  for j in range(len(matriz[i])):
                      print(matriz[i][j], end=" ")
                  print()
              
              # Salida:
              # 1 10 3
              # 4 5 6
              # 7 8 9

Ejemplo en C:
#include <stdio.h>

              int main() {
                  // Definir una matriz 3x3
                  int matriz[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};

                  // Acceder a un elemento
                  printf("%d\n", matriz[1][2]);  // Salida: 6

                  // Modificar un elemento
                  matriz[0][1] = 10;
                  printf("%d\n", matriz[0][1]);  // Salida: 10

                  // Recorrer la matriz
                  for (int i = 0; i < 3; i++) {
                      for (int j = 0; j < 3; j++) {
                          printf("%d ", matriz[i][j]);
                      }
                      printf("\n");
                  }

                  // Salida:
                  // 1 10 3
                  // 4 5 6
                  // 7 8 9

                  return 0;
              }

Ejemplo en JavaScript:
// Definir un array bidimensional
              let matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];

              // Acceder a un elemento
              console.log(matriz[1][2]);  // Salida: 6

              // Modificar un elemento
              matriz[0][1] = 10;
              console.log(matriz[0][1]);  // Salida: 10

              // Recorrer el array bidimensional
              matriz.forEach(fila => {
                  fila.forEach(elemento => {
                      console.log(elemento);
                  });
              });

              // Salida:
              // 1
              // 10
              // 3
              // 4
              // 5
              // 6
              // 7
              // 8
              // 9

Tipos de Arrays Bidimensionales:
1. Arrays de arrays: Cada fila es un array independiente que contiene sus propios elementos.
2. Matrices: El array bidimensional se puede ver como una tabla de datos.

Desventajas de los Arrays Bidimensionales:
- Espacio en memoria: Los arrays bidimensionales pueden ocupar más memoria en comparación con los arrays unidimensionales.
- Complejidad en el manejo: Las operaciones en arrays bidimensionales suelen ser más complejas debido a la necesidad de manejar dos índices.

Comparación con otras estructuras de datos:
- Los arrays bidimensionales ofrecen un acceso rápido a los elementos, pero son menos flexibles que las listas enlazadas en las que el tamaño puede cambiar dinámicamente.
- Son más fáciles de usar para representar tablas de datos que las listas enlazadas, pero menos eficientes en términos de inserciones y eliminaciones de elementos.
