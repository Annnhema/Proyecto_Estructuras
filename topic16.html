Linear Search

Información sobre Linear Search

**Linear Search** es un algoritmo de búsqueda en el que se examinan cada uno de los elementos de una lista secuencialmente hasta encontrar el valor deseado o hasta llegar al final de la lista.

**Características del Linear Search:** - **Simplicidad:** Es un algoritmo sencillo de implementar. - **Búsqueda secuencial:** Requiere que se revisen todos los elementos de la lista. - **Eficiencia:** Su tiempo de ejecución es O(n), donde n es el número de elementos en la lista, lo que lo hace menos eficiente para listas grandes. **Pasos del algoritmo:** 1. Comienza desde el primer elemento de la lista. 2. Compara el valor buscado con el elemento actual. 3. Si los elementos coinciden, devuelve el índice de ese elemento. 4. Si no coincide, pasa al siguiente elemento. 5. Si se llega al final de la lista sin encontrar el valor, devuelve un valor que indica que no se ha encontrado el elemento (por ejemplo, -1). **Algoritmo en pseudocódigo:** 


                function linearSearch(arr, target):
                    for i from 0 to length(arr) - 1:
                        if arr[i] == target:
                            return i
                    return -1

Ejemplo en Python:

                def linear_search(arr, target):
                    for i in range(len(arr)):
                        if arr[i] == target:
                            return i
                    return -1

                arr = [2, 4, 6, 8, 10, 12]
                target = 8
                result = linear_search(arr, target)
                if result != -1:
                    print(f"Elemento encontrado en el índice {result}")
                else:
                    print("Elemento no encontrado")

Ejemplo en C:

                #include <stdio.h>

                int linearSearch(int arr[], int n, int target) {
                    for (int i = 0; i < n; i++) {
                        if (arr[i] == target)
                            return i;
                    }
                    return -1;
                }

                int main() {
                    int arr[] = {2, 4, 6, 8, 10, 12};
                    int target = 8;
                    int n = sizeof(arr) / sizeof(arr[0]);
                    int result = linearSearch(arr, n, target);
                    if (result != -1)
                        printf("Elemento encontrado en el índice %d\n", result);
                    else
                        printf("Elemento no encontrado\n");
                    return 0;
                }

Ejemplo en JavaScript:

                function linearSearch(arr, target) {
                    for (let i = 0; i < arr.length; i++) {
                        if (arr[i] === target)
                            return i;
                    }
                    return -1;
                }

                let arr = [2, 4, 6, 8, 10, 12];
                let target = 8;
                let result = linearSearch(arr, target);
                if (result !== -1) {
                    console.log("Elemento encontrado en el índice " + result);
                } else {
                    console.log("Elemento no encontrado");
                }

**Ventajas de Linear Search:** - Fácil de entender e implementar. - No requiere que la lista esté ordenada. **Desventajas de Linear Search:** - Ineficiente para listas grandes, ya que tiene que recorrer todos los elementos en el peor de los casos. - Su complejidad temporal es O(n), lo que significa que el tiempo de ejecución crece linealmente con el tamaño de la lista.
