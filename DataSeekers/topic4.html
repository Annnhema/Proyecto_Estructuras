Queues using Arrays

Información sobre Colas Usando Arrays

Definición:
Una cola (queue) es una estructura de datos en la que los elementos se insertan al final (enqueue) y se extraen desde el principio (dequeue), siguiendo la política FIFO (First In, First Out). Usualmente, las colas se implementan usando arrays o listas enlazadas.

Operaciones comunes:
1. Enqueue: Agregar un elemento al final de la cola.
queue.enqueue(elemento)

2. Dequeue: Eliminar el primer elemento de la cola.
queue.dequeue()

3. Peek: Obtener el primer elemento sin eliminarlo.
queue.peek()

4. IsEmpty: Verificar si la cola está vacía.
queue.isEmpty()

Ejemplos:
Ejemplo en Python:

# Implementación de una cola usando un array
              class Queue:
                  def __init__(self):
                      self.queue = []
                  
                  def enqueue(self, item):
                      self.queue.append(item)
                  
                  def dequeue(self):
                      return self.queue.pop(0)
                  
                  def peek(self):
                      return self.queue[0] if self.queue else None
                  
                  def isEmpty(self):
                      return len(self.queue) == 0
              
              # Crear una cola
              q = Queue()
              q.enqueue(1)
              q.enqueue(2)
              q.enqueue(3)
              
              # Operaciones
              print(q.dequeue())  # Salida: 1
              print(q.peek())  # Salida: 2

Ejemplo en C:
#include <stdio.h>

              #define MAX 5
              int queue[MAX];
              int front = -1, rear = -1;

              // Enqueue
              void enqueue(int item) {
                  if (rear == MAX - 1) {
                      printf("Queue is full!\n");
                  } else {
                      if (front == -1) front = 0;
                      rear++;
                      queue[rear] = item;
                  }
              }

              // Dequeue
              int dequeue() {
                  if (front == -1 || front > rear) {
                      printf("Queue is empty!\n");
                      return -1;
                  } else {
                      int item = queue[front];
                      front++;
                      return item;
                  }
              }

              // Main
              int main() {
                  enqueue(1);
                  enqueue(2);
                  enqueue(3);
                  printf("%d\n", dequeue());  // Salida: 1
                  return 0;
              }

Ejemplo en JavaScript:
// Implementación de una cola usando un array
              class Queue {
                  constructor() {
                      this.queue = [];
                  }

                  enqueue(item) {
                      this.queue.push(item);
                  }

                  dequeue() {
                      return this.queue.shift();
                  }

                  peek() {
                      return this.queue[0];
                  }

                  isEmpty() {
                      return this.queue.length === 0;
                  }
              }

              // Crear una cola
              const q = new Queue();
              q.enqueue(1);
              q.enqueue(2);
              q.enqueue(3);

              // Operaciones
              console.log(q.dequeue());  // Salida: 1
              console.log(q.peek());  // Salida: 2

Desventajas:
- El tamaño de la cola es limitado en arrays de tamaño fijo.
- Las operaciones de encolado y desencolado pueden ser costosas si no se implementan correctamente.
