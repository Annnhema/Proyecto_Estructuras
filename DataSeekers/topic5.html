List Data Structures

Información sobre Listas

Definición:
Una lista es una estructura de datos lineal que almacena una colección de elementos, en la cual los elementos pueden estar de forma contigua o no en la memoria. Las listas pueden ser de diferentes tipos: listas enlazadas, listas doblemente enlazadas, listas circulares, entre otras.

Operaciones comunes:
1. Agregar un elemento: Insertar un elemento en la lista.
list.append(elemento)

2. Eliminar un elemento: Eliminar un elemento de la lista.
list.remove(elemento)

3. Buscar un elemento: Buscar un elemento en la lista.
list.index(elemento)

4. Recorrer la lista: Iterar sobre todos los elementos de la lista.
for elem in list:
     print(elem)

Ejemplos:
Ejemplo en Python:

# Definición de una lista
              lista = [1, 2, 3, 4, 5]
              
              # Agregar un elemento
              lista.append(6)
              
              # Eliminar un elemento
              lista.remove(3)
              
              # Buscar un elemento
              print(lista.index(4))  # Salida: 2
              
              # Recorrer la lista
              for elem in lista:
                  print(elem)

Ejemplo en C:
#include <stdio.h>
              #include <stdlib.h>

              typedef struct Node {
                  int data;
                  struct Node* next;
              } Node;

              // Función para agregar un elemento al final
              void append(Node** head_ref, int new_data) {
                  Node* new_node = (Node*)malloc(sizeof(Node));
                  Node* last = *head_ref;
                  new_node->data = new_data;
                  new_node->next = NULL;

                  if (*head_ref == NULL) {
                      *head_ref = new_node;
                      return;
                  }
                  while (last->next != NULL) {
                      last = last->next;
                  }
                  last->next = new_node;
              }

              // Función para imprimir la lista
              void printList(Node* node) {
                  while (node != NULL) {
                      printf("%d -> ", node->data);
                      node = node->next;
                  }
                  printf("NULL\n");
              }

              int main() {
                  Node* head = NULL;
                  append(&head, 1);
                  append(&head, 2);
                  append(&head, 3);
                  printList(head);
                  return 0;
              }

Ejemplo en JavaScript:
// Definir una lista (Array en JS)
              let list = [1, 2, 3, 4, 5];
              
              // Agregar un elemento
              list.push(6);
              
              // Eliminar un elemento
              list.splice(list.indexOf(3), 1);
              
              // Buscar un elemento
              console.log(list.indexOf(4));  // Salida: 2
              
              // Recorrer la lista
              list.forEach(elem => console.log(elem));

Desventajas:
- Las operaciones de inserción y eliminación pueden ser costosas si no se usa una implementación eficiente, como las listas enlazadas.
