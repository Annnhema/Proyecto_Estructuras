Arrays and Functions

Información sobre Arrays y Funciones

Definición:
Un array es una estructura de datos que almacena una colección de elementos del mismo tipo, mientras que una función es un bloque de código reutilizable que realiza una tarea específica. Las funciones pueden tomar arrays como argumentos y devolver arrays como resultado. Las funciones sobre arrays son comunes y permiten manipular los datos almacenados.

Funciones comunes sobre Arrays:
1. Acceso: Obtener el valor de un elemento en el índice i.
array[i]

2. Modificación: Cambiar el valor de un elemento en el índice i.
array[i] = nuevo_valor

3. Recorrido: Iterar sobre todos los elementos.
for elem in array:
     print(elem)

4. Inserción: Insertar un elemento en una posición específica.
array.insert(i, valor)

5. Eliminación: Eliminar un elemento en una posición específica.
del array[i]

6. Ordenamiento: Ordenar los elementos en el array.
array.sort()

7. Búsqueda: Buscar un elemento en el array.
if valor in array:
     print("Elemento encontrado")

8. Funciones Personalizadas: Crear funciones para operaciones más complejas sobre arrays.

Ejemplos:
Ejemplo en Python:

# Definición de un array
              arr = [1, 2, 3, 4, 5]

              # Función para recorrer e imprimir el array
              def recorrer_array(array):
                  for num in array:
                      print(num)

              # Llamada a la función
              recorrer_array(arr)  # Salida: 1 2 3 4 5

              # Función para modificar un elemento
              def modificar_array(array, index, nuevo_valor):
                  array[index] = nuevo_valor

              # Llamada a la función
              modificar_array(arr, 2, 10)
              print(arr)  # Salida: [1, 2, 10, 4, 5]

Ejemplo en C:
#include <stdio.h>

              void recorrer_array(int array[], int size) {
                  for (int i = 0; i < size; i++) {
                      printf("%d\n", array[i]);
                  }
              }

              void modificar_array(int array[], int index, int nuevo_valor) {
                  array[index] = nuevo_valor;
              }

              int main() {
                  int arr[5] = {1, 2, 3, 4, 5};
                  recorrer_array(arr, 5);  // Salida: 1 2 3 4 5

                  modificar_array(arr, 2, 10);
                  printf("%d\n", arr[2]);  // Salida: 10
                  return 0;
              }

Ejemplo en JavaScript:
// Definir un array
              let arr = [1, 2, 3, 4, 5];

              // Función para recorrer e imprimir el array
              function recorrerArray(array) {
                  array.forEach(num => console.log(num));
              }

              // Llamada a la función
              recorrerArray(arr);  // Salida: 1 2 3 4 5

              // Función para modificar un elemento
              function modificarArray(array, index, nuevoValor) {
                  array[index] = nuevoValor;
              }

              // Llamada a la función
              modificarArray(arr, 2, 10);
              console.log(arr);  // Salida: [1, 2, 10, 4, 5]

Ventajas de usar Funciones con Arrays:
- Modularidad: Puedes dividir el código en partes más pequeñas y reutilizables.
- Claridad: Las funciones permiten que el código sea más legible y fácil de entender.
- Reutilización: Las mismas funciones pueden ser usadas en diferentes partes del programa sin duplicar código.

Desventajas:
- Rendimiento: Si no se manejan adecuadamente, el uso excesivo de funciones puede afectar el rendimiento, especialmente en lenguajes que no optimizan bien las llamadas a funciones.
- Complejidad: El uso de funciones complejas puede hacer que el código sea más difícil de seguir.

Comparación con otras estructuras de datos:
- Las funciones sobre arrays son extremadamente útiles para modularizar operaciones que se realizan de forma repetida sobre los datos.
- Son más fáciles de usar y entender que las funciones sobre listas enlazadas, ya que los arrays son de acceso directo y su manejo es más sencillo.
